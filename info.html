Asistente Virtual (chatbot)

El asistente virtual ( chatbot) es un software orientado a proporcionar ayuda a los usuarios aportando información. Este proyecto tiene como objetivo aportar  información a todos aquellos usuarios que deseen ingresar al mundo de la programación pero no saben por donde empezar, El proyecto chatbot, cuenta con la capacidad de aportar link de cursos de E-Learning sobre programación, una interfaz sencilla y facil de usar, y a su vez cuenta con un bot amigable y disponible en todo momento. Este proyecto fue creado a raíz de que muchas personas desean aprender programación pero no cuentan con idea alguna de donde pueden empezar a aprender, como también no se ha realizado una aplicación web que cuente con un asistente virtual dispuesto a ayudar a todos esos usuarios antes mencionados.
